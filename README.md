# Real-Valued Equation Solver on Raspberry Pi 3 and STM32

## Overview

This project presents the implementation and performance evaluation of real-valued equation solving algorithms on two embedded computing platforms: Raspberry Pi 3 and STM32.

The objective of the project is to investigate the differences between a single-board computer and a microcontroller when solving mathematical equations. The same algorithms are implemented independently on both platforms, allowing a direct comparison of computational performance, numerical accuracy, execution time, and hardware resource utilization.

The project was developed as part of an embedded systems study at Ho Chi Minh City University of Technology (HCMUT), Vietnam National University Ho Chi Minh City.

---

## Project Objectives
- Implement real-valued equation solving algorithms on Raspberry Pi 3.
- Implement the same algorithms on STM32.
- Solve mathematical equations and numerical problems.
- Compare execution time and computational performance.
- Evaluate numerical accuracy and stability.
- Analyze the advantages and limitations of each platform.

## Results

The developed system successfully demonstrates the implementation of identical equation-solving algorithms on both Raspberry Pi 3 and STM32 platforms.

Experimental evaluation enables direct comparison of:
- Computational performance.
- Numerical accuracy.
- Execution speed.
- Resource utilization.

The project highlights the trade-offs between a high-performance single-board computer and a resource-constrained microcontroller for mathematical computation and numerical analysis applications.
